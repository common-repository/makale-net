Makale.net wordpress eklentisi satın aldığınız içerikleri hızlı ve kolayca Wordpress sitenizde yayınlamanızı sağlar. Eklenti üzerinden yeni seo uyumlu hazır makaleler satın alabilir veya ihtiyaçlarınıza uygun makale siparişi verebilirsiniz.

== Description ==
Makale.net Wordpress eklentisi Makale.net üzerinden verilen içerik siparişlerinizi Wordpress panelinizden kolayca yönetmenizi sağlar. Eklenti sayesinde tüm içerikleri tek tık ile hemen sitenizde yayınlayabilirsiniz. İhtiyaçlarınıza göre yeni içerik siparişi verebilirsiniz. 40+ farklı kategoride binlerce hazır makaleden satın alabilir sitenizde yayınlayabilirsiniz. Eklenti sayesinde ayrıca toplu taslak olarak kaydetme, toplu yayınlama, kullandım / kullanmadım özellikleri ve bakiye yükleme imkanı da bulunmaktadır. 

== Installation ==

= Nasıl Kullanabilirim ? =
1. Makale.net eklentisini indirin, kurun ve eklentiyi aktif edin.
2. [Buraya tıklayarak](https://www.makale.net/entegrasyon/wordpress) API-KEY alanından benzersiz bir API anahtarı oluşturun.
3. Eklenti menüsünde yer alan Ayarlar bölümüne giriniz.
4. Makale.net kullanıcı adı ve oluşturduğunuz API-KEY’i giriniz.
5. İşlemler tamamlandı! Artık eklentiyi kullanarak tüm işlemlerinizi yapabilirsiniz.

== Frequently Asked Questions ==

= Nasıl Kurulur? =

İndirdiğiniz eklenti dosyalarını sitenizin /wp-content/plugin/ dizinine yükleyin, ardından eklentiler sayfasından eklentiyi etkinleştirin.

= Makale.net Nedir? =

Makale.net sitenizin ihtiyaç duyduğu içerikleri sipariş verebileceğiniz, uzman yazarlar tarafından içeriklerinizin hazırlandığı, editörlerin kontrol ve teslim ettiği bir içerik servisidir. Tamamen özgün, SEO uyumlu ve nitelikli içerikler üretir. Ayrıca 40+ farklı kategoride kişiye özel satışa sunulan hazır içerikler de satın alarak hemen kullanabilirsiniz. 

= Eklenti İle Yapılabilecekler =

* Satın aldığınız içerikleri tek tık ile sitenizde yayınlayabilirsiniz.
* Hazır makale satın alarak hemen sitenize ekleyebilirsiniz. 
* Toplu ve tekli olarak içerikler için işlem yapabilirsiniz. 
* İçerikleri taslak olarak kaydederek daha sonra yayınlayabilirsiniz. 


== Screenshots ==
1. Kullanıcı adı alanına Makale.net kullanıcı adınızı giriniz. API Durumu: Api anahtarınızın aktif / pasif durumunu gösterir. API-KEY alanına Makale.net üzerinden almış olduğunuz API-KEY giriniz. 
2. Makale siparişi verme sayfası üzerinden istekleriniz ve ihtiyaçlarınız doğrultusunda makale siparişi verebilirsiniz. Eğer içeriklerinizin 24 saat içerisinde teslim edilmesini istiyorsanız “Express Makale” hizmetinden de yararlanabilirsiniz.
3. Verdiğiniz makale siparişlerinizi Makale Siparişlerim menüsü üzerinden takip edebilmektesiniz. Siparişinizin durumunu takip edebilir, hazırlanmış makaleleri siparişinizin tamamlanmasını beklemeden hemen kullanabilirsiniz. 
4. Sitenize uygun hazır makaleler satın alarak hemen yeni içerikleri sitenize ekleyebilirsiniz. 40+ farklı kategoride binlerce hazır makale bulunmaktadır. Özgün ve SEO uyumlu hazır makaleler sadece bir kişiye satılmaktadır ve satın alınan hazır makale sistemden otomatik olarak kaldırılmaktadır.
5. Satın aldığınız hazır makaleler için tekli veya toplu olarak işlem yapabilirsiniz. Sitenize “Hemen Yayımla” diyerek ekleyebilir veya daha sonra yayınlamak için “Taslak Olarak Kaydet” seçeneğini kullanabilirsiniz. İçerikleri “kullandım / kullanmadım” olarak işaretleyerek hataları önleyebilirsiniz.
